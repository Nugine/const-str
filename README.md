## Develop

Toolchain

+ [Rust 1.64.0 or newer](https://rustup.rs/)
+ [just](https://github.com/casey/just)

Get the source code

```bash
git clone https://github.com/Nugine/const-str.git
cd const-str
```

#### Run checks and tests

```bash
just dev
```

#### Open documentation

```bash
just doc
```
